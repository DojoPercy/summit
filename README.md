# The Technology Boardroom Platform

An exclusive, high-end "Digital Editorial" platform for C-suite technology leaders and executive councils. Built with Next.js, Framer Motion, and Tailwind CSS.

## 🏛️ Design Philosophy: Digital Editorial

The platform is designed to mirror a premium business journal or high-end editorial publication.

- **Typography**: sophisticated pairing of serif (Lora) and display (Playfair Display) fonts.
- **Visuals**: Immersive, full-screen imagery with glassmorphism and grayscale-to-color transitions.
- **Pacing**: Staggered grids and alternating split-layouts to provide a rhythmic, executive reading experience.

## 🚀 Key Features

### Homepage

- **Executive Manifesto**: A dual-column authority narrative.
- **Strategic Pillars**: Staggered grid of interactive, image-backed leadership categories.
- **Global Showcase**: Alternating editorial features for Dubai, Riyadh, Accra, and NYC.
- **Impact Stats**: Data-driven social proof section for board-level trust.

### About Page

- **Membership Blueprint**: Sticky secondary navigation for detailed benefits.
- **Persona Cards**: High-end visuals highlighting the community's seniority.
- **Executive Profile Showcase**: Immersive visual of the digital member experience.

### Projects Page

- **Platform Showcase**: An image-driven journey through Roundtables, AI Dialogues, and Fintech hubs.
- **Strategic Focus Grids**: Quick-scan value propositions per platform.

### Contact Page

- **Executive Engagement Portal**: prestigious lead capture with a focus on trust and confidentiality.
- **Global Office Grid**: Professional regional presence layout.

## 🛠️ Technical Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Theming**: next-themes (Dark mode optimized)

## 📸 Image Integration

All visual assets are centralised in `lib/images.ts`. To update the platform's imagery:

1. Download or generate your professional high-resolution assets.
2. Place them in the `public/images/` directory.
3. Update the corresponding key in `lib/images.ts`.

## ⚙️ Development

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build
```
